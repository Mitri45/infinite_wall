# Infinite Wall submission video

Target length: 90 seconds. Hard limit: under 3 minutes.

## Before recording

- Install the final GitHub Release build and fully quit any older tray process.
- Confirm the header says Codex is ready.
- Use a clean desktop with no notifications, private filenames, terminals, or
  unrelated applications visible.
- Select **Nature** in **Infinite** mode and use a prepared generation so the
  recording can cut across the one-to-two-minute wait.
- Record at 1080p with the pointer enlarged if possible. Record voice cleanly;
  do not add copyrighted music.

## Shot list and narration

### 0:00–0:10 — Launch and product promise

**Screen:** Launch Infinite Wall from the Applications menu. Let the category
library and Codex-ready status settle.

**Say:**

> This is Infinite Wall, a local-first desktop wallpaper generator built with
> Codex and GPT-5.6. It creates original wallpapers, keeps a private local
> library, and applies them directly to my desktop.

### 0:10–0:25 — Choose a direction

**Screen:** Select **Nature**. Briefly show Infinite, Curated, and Custom, then
return to Infinite.

**Say:**

> I start with one of thirteen visual worlds. I can ask for a fresh concept,
> choose a curated scene, or provide my own direction. Here I will let Codex
> invent something new within the Nature theme.

### 0:25–0:43 — Generate with Codex

**Screen:** Click **Generate wallpaper**. Show the stage-based progress for a few
seconds, then hard-cut to the completed result. Add a small edit caption:
**Generated in about 1–2 minutes**.

**Say:**

> Infinite Wall sends the theme, composition guidance, and display size to my
> signed-in Codex CLI. GPT-5.6 turns that direction into the final image prompt
> and generates the wallpaper in the same isolated run. The app needs no
> separate API key or cloud backend.

### 0:43–0:58 — Review and apply

**Screen:** Pause on the generated title and preview, click **Apply wallpaper**,
then reveal the changed desktop.

**Say:**

> The result is validated, saved locally, and shown for approval before my
> desktop changes. One click applies it, while the history remains available for
> reuse, favorites, or removal.

### 0:58–1:10 — Scheduling and tray

**Screen:** Briefly show Settings with the schedule status, then the tray menu
with **Generate Current Direction** and **Surprise Me — Random Theme**.

**Say:**

> Optional scheduling can create and apply new wallpapers automatically, and
> the tray provides quick generation and schedule controls without reopening the
> full workflow.

### 1:10–1:30 — Build provenance and close

**Screen:** Return to the main category view, then end on the app name and final
wallpaper.

**Say:**

> Codex also collaborated on building Infinite Wall: translating the product
> direction into Electron and TypeScript, prototyping the interface, writing
> tests, packaging Linux releases, and resolving review feedback. I directed the
> product decisions and validated every packaged flow locally. Infinite Wall
> turns Codex into a calm, practical creative tool for the desktop.

## Recording notes

- Do not wait through generation in real time; use one clean hard cut.
- Keep the video public on YouTube and verify it while signed out.
- Use a plain title such as **Infinite Wall — OpenAI Build Week demo**.
- In the YouTube description, include the public repository URL and note that
  the downloadable Linux build is available from GitHub Releases.
